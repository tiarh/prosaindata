Getdata#

Get data : ambil data dari beberapa sumber skenario#

A. Sepal length : dari posgresql local

B. sepal width : di postgeql elephantsql

C. petal length : di mysql local

D.petal width +class : di sql server

Langkah-langkah:#

  1. Install library SQLAlchemy, Pandas, psycopg2-binary, pyodbc, mysql-connector-python

  2. import semua library yang diperlukan

  3. buat koneksi ke masing-masing database dengan SQLAlchemy

  4. ambil data dari table dengan skenario seperti diatas

  5. membaca data dari berbagai database kedalam dataframe

  6. menggabungkan data dari berbagai database dengan menggunakan concat
